# Retime - 个人主页模拟器

仿易班校园社交平台 UI 的个人主页 App，支持自定义资料、动态卡片、标签管理等。

## 功能

- 📱 个人主页展示：封面图、头像、昵称、学校/学院/班级信息
- 📝 内容卡片：学习打卡、劳动打卡等动态卡片，可编辑标题、署名、标签、图片
- 🏷️ 标签管理：预设标签 + 自定义标签，灵活分类
- 🏫 一键宿舍打卡：自动生成整月的宿舍打卡记录
- 🔒 动态口令验证：每日口令保护设置入口
- 💾 本地持久化：SharedPreferences 保存所有数据，无需服务器
